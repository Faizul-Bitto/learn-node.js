## ğŸš€ Getting Started

### Prerequisites
- Node.js (v14 or higher recommended)
- npm (comes with Node.js)
- Basic JavaScript knowledge

### Installation

Each module has its own `package.json`. Navigate to the specific folder and install dependencies:

```bash
cd "1. Node.js/a. global object"
npm install
```

### Running Examples

Most modules include example code. Run them with:

```bash
node index.js
```

For Express.js projects with nodemon:

```bash
npm start
```
